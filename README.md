# **MonoCraft – A Minimal Portfolio Website**

MonoCraft is a clean, minimal, single-page portfolio website built using **HTML** and **CSS**. The project focuses on clean structure, modern styling, and a clear presentation of work and ideas.

The design focuses on readability, whitespace, and subtle interactions—inspired by modern developer portfolios.


## 🎯 Purpose of This Project

* Present a clean, minimal personal portfolio layout
* Demonstrate semantic HTML and modern CSS practices
* Showcase projects and written articles in a structured way
* Provide a strong foundation for a personal or developer portfolio website


## ✨ Features

* Minimal, modern UI design
* Semantic HTML5 structure
* Clean typography using **Google Fonts (Poppins)**
* Icon support via **Remix Icons**
* Project and article showcase sections
* Newsletter call-to-action section
* Simple, readable CSS architecture
* Fully static (no JavaScript)


## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Custom properties, Flexbox)
* **Google Fonts**
* **Remix Icon CDN**

## 🚀 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/codebysiam/minimal-portfolio.git
   ```
2. Navigate into the project folder:

   ```bash
   cd minimal-portfolio
   ```
3. Open `index.html` in your browser:

   * Double-click the file, or
   * Use Live Server (VS Code extension)
